# Console Nexus
An open-source manager for your emulators and game libraries.

## Features
Until v1.0.0 these features are all just planned. See the roadmap below to see progress. v1.0.0 will be full but basic Xenia support

### Core
- Add and manage multiple emulators
- Launch emulators and games
- Scan game directories and display libraries

### Emulator Management
- Create emulator profiles
- Backup emulator profiles
- Support multiple emulator versions
- Built-in installation support for selected emulators
- Manage launch arguments and settings per emulator profile

### Game Management
- Organise games per emulator and profile
- Manual game addition and removal
- Basic game metadata support

### Quality of Life
- Searchable game library
- PySide6 GUI

## Supported Emulators

### Xbox 360
- [Xenia](https://github.com/xenia-project)
- [Xenia Canary](https://github.com/xenia-canary)

## Tech Stack
- [Python](https://www.python.org) 3.11.6
- [PySide6](https://doc.qt.io/qtforpython-6/)
- [PyGithub](https://github.com/PyGithub/PyGithub)
